## 真实用户怎么说：吐槽和常见抱怨

我不想只提供片面的观点。以下是我发现的批评意见，以及背景说明。

- **“我被封锁了好几次。”** 这条 Reddit 评论提到了一件事。背景是：这位用户使用 BandwagonHost 进行特定流量模式的服务，导致他们的 IP 在某些网络下被标记。这并非 BandwagonHost 特有的可靠性问题，而是你需要注意的——如果使用场景涉及规避检测的流量，某些数据中心 IP 会被区域封锁。官方对此表示 CN2 GIA 网络对 DDoS 的容忍度有限，可能需要 IP 空路由。因此，如果遭受针对性攻击，你的服务会暂时下线。

- **“自管理模式意味着你需要自己动手。”** 这不是 Bug，而是价格的决定因素。BandwagonHost 负责硬件、网络和基础设施。你负责 OS 层以上的一切。如果你需要 cPanel、托管式 WordPress 或有人通过电话指导你配置 Apache，这不是合适的主机。反复出现的批评是：“我以为我得到了托管支持，但我没有。”在购买前请仔细阅读自管二字。

- **“网站看起来过时了。”** 这是真的，2012 年的审美依然在线。但说实话，我宁愿要一个 2012 年的网站配上 2026 年的服务器，而不是反过来。大多数长期用户不在乎这一点。

- **“库存波动令人沮丧。”** 限量版套餐（如 "THE PLAN"、"MINICHICKEN"、"BiggerBox Pro" 系列）可以以常规套餐三分之一到五分之一的价格出售。但它们很快售罄，经常缺货。社区存在库存监控工具正是因为这个原因。如果你盯上了限量补货，准备好在通知发出时抢购。

## 网络解释：为什么 CN2 GIA 在用户评价中占据主导地位

如果你读过任何 BandwagonHost 的用户评价，"CN2 GIA" 这个词一定会出现。所以让我解释一下为什么它很重要——因为这是该公司获得最多赞誉的部分。

中国电信提供四个层级的 IP 转接：

1. **AS4134 (ChinaNet/163 Net)** — 最常见，最便宜，但在高峰时段严重拥堵，丢包率超过 30%。
2. **AS4809 CN2 GT (Global Transit)** — 原本设计为解决拥堵的方案，但自 2019 年以来，几乎与 AS4134 一样拥堵，尽管成本更高。
3. **AS4809 CN2 GIA (Global Internet Access)** — 最昂贵，但最稳定。即使在晚高峰，丢包率也接近零。这是你用于 Web 会议、VOIP、为主机中国受众服务内容、在线游戏等场景的网络。CN2 GIA IP 转接价格可能高达每兆比特 120 美元——相当于在部分市场上 1 Gbps 连接每月 100,000 美元的账单。
4. **AS23764 CTGNet** — 中国电信的最新添加，在实际应用中等同于 CN2 GIA 的定价和性能。

BandwagonHost 在洛杉矶通过两个数据中心运营 8 条 10 Gbe CN2 GIA/CTGNet 链路。USCA_9 数据中心将所有去往中国的流量发送至三个运营商：CN2 GIA (中国电信 AS4809)、CMIN2 (中国移动 AS58807) 和中国联通高级 (AS10099)。此外还有与 Google 和其他本地运营商的直接对等连接。

这就是为什么用户评价始终称赞网络稳定性的原因：该公司确实花钱购买了大多数竞争对手不会购买的优质转接，因为它贵得离谱且难以获得。
